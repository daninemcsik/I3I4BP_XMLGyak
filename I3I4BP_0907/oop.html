Szoftvertechnológia
Objektum-orientált programozás
A tárgy célja az objektum-orientált programozás alapelveinek elsajátítása.
  1. A Java története, jellemzői. Java eszközök, JDK. Fejlesztési környezetek.
    A Java nyelv alap elemei (típusok, literálok, operátorok, utasítások)
  2. Objektum orientált programozás fogalma, alapelvei. Osztály definíció,
    példányosítás, hivatkozások az osztály tagjaira. Referencia változók.
  3. Hozzáférési kategóriák. Konstruktor I. Szemétgyűjtő mechanizmus.
    Példány élettartam. Adattagok. Metódusok, paraméter átadás, függvény
    túlterhelés.
  4. Csomagok (package), osztályok láthatósága, importálás. Osztályszintű
    tagok és használatuk.
  5. Speciális osztályok, tömb, enum.
  6. Öröklődés, polimorfizmus. Metódus felüldefiniálás. Konstruktor II.
  7. Referenciák statikus és dinamikus típusai. Object osztály.
  8. Final tagok. Absztrakt metódusok és osztályok. Interfészek.
  9. Interfészek. Interface referenciák.
  10. Kivétel kezelés.
  11. Beágyazott osztályok.
  12. Osztálytervezési szempontok.
